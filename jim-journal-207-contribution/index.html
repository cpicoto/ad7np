DATV software tools for reliable amateur transmit/receive

Hi Jim and BATVC readers,

For our WW7ATS Digital Amateur Television (DATV) repeater workflow, we built a practical software toolchain that connects RF DATV ingest to an operator-friendly internet viewing experience. The stack combines:

- HiDesReceiver (desktop): DVB-T DATV reception and control, feeding the streaming path.
- WW7ATS (native mobile/desktop app): a PTT (push-to-talk) transmit interface with clear connection-state UX, secure stream-key handling, and a low-latency HLS viewer.
- Azure MediaMTX + nginx (cloud headend): RTMP/RTMPS ingest and remux to HLS, with automation hooks and an API for monitoring.

Engineering emphasis: field reliability under short operator transmit sessions, deterministic headend behavior, and straightforward monitoring so net control can quickly verify “signal vs no-signal” and operator connection state.
